AIS Friends

Get one at aisfriends.com → Account → Details → API Tokens. Rate-limited to 1 request/minute.

{% set af_backend = cfg.sources.aisfriends.backend or 'flaresolverr' %}

aisfriends.com sits behind a Cloudflare JS / WebAssembly challenge that HTTP-only clients can't solve. FlareSolverr is a tiny Docker container running headless Chromium that solves the challenge for us. The other backends are no-Docker fallbacks for networks that haven't been escalated yet.

Start the container with:
docker run -d --name flaresolverr --restart unless-stopped -p 8191:8191 ghcr.io/flaresolverr/flaresolverr:latest
